# Event Registration Web

Frontend application for the Event Registration System built with React, TypeScript, and Vite.

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- REST API

## Features

- Dashboard
- Categories Management
- Events Management
- Participants Management
- Event Registrations
- Event Details

## Getting Started

### Install dependencies

```bash
npm install
```

### Configure environment

Create a `.env` file based on `.env.example`

```env
VITE_API_BASE_URL=http://localhost:5031/api
```

### Run the project

```bash
npm run dev
```

## Build

```bash
npm run build
```
